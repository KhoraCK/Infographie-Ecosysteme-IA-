Procédure générée

Titre

Imprimante inaccessible après mise à jour Windows

Symptômes
  • Imprimante réseau non détectée
  • Erreur "Le pilote n'est pas disponible"
  • Apparaît après Windows Update
Étapes de résolution
  • Supprimer l'imprimante existante (Paramètres > Imprimantes)
  • Télécharger driver universel HP (support.hp.com)
  • Installer le driver en mode administrateur
  • Réajouter l'imprimante par IP
  • Tester avec page de test
Basée sur ticket #4521
Validée par Marc D.

Diagnostic lenteur VPN site distant
Utilisateurs d'un site distant signalent des lenteurs lors de l'accès aux ressources centrales via VPN.
  1. Vérifier charge CPU du firewall (cible < 80%)
  2. Contrôler nombre de tunnels VPN actifs (limite 50)
  3. Si saturation → Redémarrer service IPSec
  4. Tester latence avec traceroute (cible < 100ms)
